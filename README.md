# Todolist

Учебный проект - todo приложение на чистом JavaScript, это доработанное за наставником приложение. Проект создан в рамках тренировки работы с Git - тренировка форка и пулл-реквеста.

### Функционал:
- Можно добавить карточку с заданием.
- Удалить карточку с заданием.
- Копировать.
- Редактировать.

### Запуск проекта:
Код моей версии приложения можно найти в ветке my-btn-code.
